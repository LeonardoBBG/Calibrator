FACTUAL PROXIMITY
- HIGH: same or very similar factual setting, such as dismissal for alleged work avoidance, technical system activity, ticketing, logs, queue work, productivity, or system-based allegations.
- MEDIUM: different factual detail but similar employment context and misconduct/investigation structure.
- LOW: different factual setting with only broad procedural or evidential overlap.
- ANALOGY_ONLY: factually different but structurally useful by analogy. Use where underlying conduct differs materially but investigation, procedure, evidence, explanation-testing, or appeal reasoning maps onto existing WS themes.
- UNKNOWN: insufficient information.

TRANSFERABILITY RATING
- HIGH: directly usable for reinforcing existing WS themes with strong structural fit.
- MEDIUM: materially useful but requires careful analogical framing.
- LOW: limited use for narrow or peripheral points.
- ANALOGY_ONLY: structural analogy only; do not imply factual similarity.
- NON_TRANSFERABLE: quarantine, not usable for WS reinforcement.
- UNKNOWN: insufficient information.

LIABILITY CONSISTENCY
- If claimant_liability_outcome is WIN, liability_outcome_strength_band must be STRONG_WIN, MODERATE_WIN, or NARROW_WIN.
- If claimant_liability_outcome is LOSS, liability_outcome_strength_band must be LOSS.
- If claimant_liability_outcome is PARTIAL, liability_outcome_strength_band must be PARTIAL.
- If claimant_liability_outcome is MIXED, liability_outcome_strength_band must be PARTIAL.
- If claimant_liability_outcome is UNKNOWN, liability_outcome_strength_band must be UNKNOWN.

REMEDY RULES
remedy_status tracks whether remedy quantum has been determined.
reduction_findings_status tracks whether Polkey and/or contributory fault reduction findings have been expressly determined in this judgment.

- If reduction_findings_status is NOT_DETERMINED, NOT_ADDRESSED, or UNKNOWN: polkey_reduction_pct and contributory_fault_pct must both be null.
- If reduction_findings_status is DETERMINED: each percentage may be numeric where expressly found, including 0, or null where that specific reduction was not addressed.
- Null means not determined, not addressed, unknown, or no express percentage. Do not infer zero.
- Zero means the Tribunal expressly found no reduction. If either percentage is 0, award_reduction_notes must make clear that zero is an express finding.
- If reduction_findings_status is DETERMINED and both percentages are 0, award_reduction_notes must confirm both zeros are express Tribunal findings and not inferred defaults.
- If any percentage is null, award_reduction_notes must explain why it is null.

SIGNAL CAUSAL WEIGHTS
signal_causal_weights must contain exactly one object per judgment signal. Each object must contain:

- signal_id: exactly one existing VALIDATED_CALIBRATION_JSON.judgment_signals[].signal_id.
- causal_weight: one of DECISIVE, CONTRIBUTING, PERIPHERAL, NEUTRAL, NEGATIVE, UNKNOWN.
- causal_weight_reason: string, at least 20 characters, tied to the Tribunal's actual reasoning.

Causal weight guidance:
- DECISIVE: primary reason for the outcome; significant reasoning or conclusive language.
- CONTRIBUTING: material supporting reason for the outcome.
- PERIPHERAL: mentioned but did not materially affect the outcome.
- NEUTRAL: no directional causal weight.
- NEGATIVE: associated with a finding adverse to claimant-side use.
- UNKNOWN: cannot be determined.
